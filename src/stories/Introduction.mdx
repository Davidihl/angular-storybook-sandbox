## Setup

When browsing through code snippets, the components need to be added to the template file (`*.component.html`).

As commentary, you will find the necessary imports in your `*.component.ts`.

E.g.:

```js filename="*.component.ts"
import { MatButtonModule } from "@angular/material/button";
```
